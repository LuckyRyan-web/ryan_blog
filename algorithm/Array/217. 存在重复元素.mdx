## 思路

首先对数组进行排序，然后对数组进行排序，遍历数组对 N 和 N+1 项进行比较，如果相等的话就是有重复的数据

## 复杂度分析

-   时间复杂度：O(N log N)，其中 N 为遍历 nums 的长度， logN 为 nums.sort 的消耗

-   空间复杂度：O(log N)，也就是 nums.sort 的复杂度
